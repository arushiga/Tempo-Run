# Tempo

CIS 4120 Assignment 5: Technical Requirements Implementation

## Tech Stack

- Swift
- SwiftUI
- Firebase

## Instructions to Run the Program

1. Open `Tempo-Run/Tempo/Tempo.xcodeproj` in Xcode.
2. Add the Firebase configuration information to `Tempo-Run/Tempo/Tempo/Resources/GoogleService-Info` using the values listed in the submission document.
3. Click the Run button in Xcode to build and launch the app.
